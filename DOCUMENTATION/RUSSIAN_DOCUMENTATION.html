1. Обзор проекта

GamePyers — это платформа для игровой сообщества, где пользователи могут просматривать посты, новости и обновления, связанные с играми. На сайте есть такие функции, как регистрация пользователей, управление постами и загрузка игр.

3. Используемые технологии

  • Python 3.11
  • Веб-фреймворк Flask
  • База данных SQLite
  • HTML, CSS, JavaScript для фронтенда
  • Git для контроля версий
  • Flask-Mail для отправки электронной почты
  • Модуль random для генерации случайных значений
  • Самодельный класс DatabaseManager для управления базой данных с подробной документацией по методам

4. Инструкции по установке

Чтобы настроить проект локально, выполните следующие шаги:

  1. Клонируйте репозиторий: git clone https://github.com/your-repo/GamePyers.git
  2. Перейдите в каталог проекта: cd GamePyers
  3. Создайте и активируйте виртуальное окружение:
    4. python -m venv venv
source venv/bin/activate  # В Windows: venv\Scripts\activate
  4. Установите зависимости: pip install "name"
  5. Запустите приложение: python run.py

5. Ключевой функционал

  • Аутентификация пользователей: регистрация, вход и управление сессиями на основе куки-файлов.
  • Управление постами: пользователи могут создавать, редактировать и удалять посты.
  • Новости и обновления: раздел новостей для получения последних обновлений по играм.
  • Загрузка игр: пользователи могут загружать игры через специальный раздел. (в будущих обновлениях)
