1. Project Overview

GamePyers is a gaming community platform where users can view game-related posts, news, and updates. The site includes features such as user registration, post management, and game downloads.

3. Technologies Used

  • Python 3.11
  • Flask Web Framework
  • SQLite Database
  • HTML, CSS, JavaScript for Frontend
  • Git for Version Control
  • Flask-Mail for email communication
  • random module for generating random values
  • Custom DatabaseManager class for managing the database with detailed documentation on methods

4. Setup Instructions

To set up the project locally, follow these steps:

  1. Clone the repository: git clone https://github.com/your-repo/GamePyers.git
  2. Navigate to the project directory: cd GamePyers
  3. Create and activate a virtual environment:
    4. python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
  4. Install dependencies: pip install "name"
  5. Run the application: python run.py

5. Key Functionality

  • User Authentication: Registration, login, and cookie-based session management.
  • Post Management: Users can create, edit, and delete posts.
  • News and Updates: A news section to stay updated on the latest game-related content.
  • Game Downloads: Users can download games from a dedicated section. (in future updates)
